Smithing

Equipment

Smithing is a gameplay mechanic originally added in 1.16 for creating Netherite tools, but later was added onto in 1.20, with the smithing templates. Earth & Water adds new smithing templates to further enchance this feature.

Steel Upgrade

The Steel Upgrade modifies a standard vanilla shield, allowing it to deal damage to entities that attack the wielder while blocking.

The smithing recipe combines a shield with a steel ingot and the Steel Upgrade to create the Spiked Shield.

Block Smithing Template

Set of Armor with Block Trim applied to it.
The smithing recipe combines a Block Smithing Template with an Iron Chestplate and a Copper Ingot to create the trimmed version of the armor.

Guard Smithing Template

Set of Armor with Guard Trim applied to it.
The smithing recipe combines a Guard Smithing Template with an Iron Chestplate and a Lapis Lazuli to create the trimmed version of the armor.
