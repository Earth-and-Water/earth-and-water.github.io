Reinforced Spawner

The Reinforced Spawner is an advanced spawner block that generates waves of mobs when activated with a Reinforced Key. Unlike regular spawners, it features a sophisticated wave-based spawning system that scales with the number of nearby players and displays the contained mob entity.

Obtaining

Structure Generation

Reinforced Spawners can be found naturally generating in the Ancient Ruins and Conduit Monument structure.

Breaking

Reinforced Spawners are quite difficult to mine and have extremely high blast resistance (50), making them resistant to most forms of destruction. They cannot be moved by pistons, and they do not drop anything upon breaking.

Usage

Configuration

Before activation, a Reinforced Spawner must be configured with a mob type:

  • Right-click the spawner with any Spawn Egg to set the mob type
  • The spawner will display the configured mob rotating inside
  • The mob type can be changed at any time before or after activation with another spawn egg
Activation with Reinforced Key

Once configured, the spawner can be activated and deactivated using a Reinforced Key:

  • Keyhole Detection: When a player holding a Reinforced Key comes within 3 blocks, a keyhole visual appears on the spawner
  • Deactivation: Right-click an active spawner with a key to deactivate it (grants advancement)
  • Activation: Right-click with the key while the keyhole is visible to activate the spawner
  • Keys are consumed on use (except in creative mode)
Waterlogging

Reinforced Spawners can be waterlogged, allowing them to function underwater.

Mechanics

Wave System

When active, the Reinforced Spawner operates on a sophisticated wave-based system:

  • Player Detection: Scans for non-creative, non-spectator players within 14 blocks
  • Wave Size: Base wave size is 3 mobs, plus 2 additional mobs per extra player (e.g., 3 mobs for 1 player, 5 for 2 players, 7 for 3 players)
  • Wave Delay: After completing a wave, there is a 60-tick (3 second) delay before the next wave starts
  • Infinite Waves: As long as the reinforced spawner is active, waves will continue spawning
Redstone Properties

Reinforced Spawners do not emit redstone signals and cannot be controlled by redstone.
